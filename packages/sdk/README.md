# @go2asia/sdk

Auto-generated API SDK from OpenAPI specification.

## Usage

```typescript
import { getUsers, createUser } from "@go2asia/sdk";

const users = await getUsers();
```

## Generation

SDK is generated from `docs/openapi/openapi.yaml`:

```bash
pnpm gen:sdk
```

This package should not be edited manually. All SDK code comes from the OpenAPI specification.
